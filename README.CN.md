# Touhou Little Maid × Hostile Neural Networks 联动模组

让车万女仆能够像玩家一样使用深度学习器，自动积累对应生物的模型进度。

## 功能简介

- **女仆也能用深度学习器**：将装有模型的深度学习器放入女仆的背包，女仆在击杀对应生物时，该深度学习器内的模型进度会逐渐增加，效果与玩家亲自使用完全一致。
- **无缝集成**：无需复杂操作，只需通过女仆的背包界面即可装备深度学习器。


## 注意事项

- 原版 Hostile Neural Networks 已存在“宠物击杀增加玩家物品栏中深度学习器进度”的机制。本联动模组在此基础上额外实现了女仆背包中深度学习器的进度积累功能，两者互不影响，可同时生效（即女仆击杀生物时，既可能增加玩家物品栏中同类型的深度学习器进度，也会增加女仆背包中的进度）。
  

## 兼容性

- **需要前置模组**：  
  - [车万女仆（Touhou Little Maid）]  
  - [Hostile Neural Networks]  
- 支持 Minecraft 版本：1.21.1 neoforge
